<h2 align="left">Installation</h2>

###

<h4 align="left">Setup</h4>

###

```
git clone https://github.com/rudymohammadbali/Whisper-Transcriber.git
```
```
cd Whisper-Transcriber
```
```
python setup.py
```
and Finally run it
```
python main.py
```
